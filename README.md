## 3 分钟快速体验

1. 进入「会员中心」切换身份（访客 / 普通用户 / VIP）
2. 到「登录管理」添加本地演示账号
3. 在「发布管理」同步草稿并执行手动发布
4. 开启自动发布，观察日志与发布记录变化

你会直观看到：公众号与订阅号运营，如何从“手动重复”升级到“自动化执行”。
