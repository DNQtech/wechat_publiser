使用帮助

纯前端版保留文案和规则说明，用于静态展示与方案演示。

快速开始
1

切换身份

先到会员中心选择访客、普通用户或 VIP，演示不同权限。

2

添加 mock 账号

普通用户和 VIP 可以在登录管理中添加本地微信账号。

3

查看草稿与发布

进入发布管理后，同步草稿、手动发布和自动发布都只在前端本地执行。

支持范围
  • 支持纯前端页面演示、功能走查、视觉评审
  • 支持本地 mock 账号、mock 草稿、mock 发布日志
  • 不接入微信支付、不请求后端接口、不依赖 Electron API
  • 打开本地 HTML 即可运行
身份说明
  • 登录-访客：只能浏览，不能操作发布功能
  • 试用激活-普通用户：可使用全部前端演示功能
  • 付费套餐激活-VIP：同样可使用全部功能，身份高亮显示
  • 到期时间与剩余时长按北京时间展示
